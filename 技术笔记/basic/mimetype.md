# mimetype
## Standards tree
>type "/" subtype ["+" suffix] *[";" parameter]
## Vendor tree
>type "/" "vnd." subtype ["+" suffix] *[";" parameter]
## Personal or vanity tree
>type "/" "prs." subtype ["+" suffix] *[";" parameter]
## Unregistered tree
>type "/" "x." subtype ["+" suffix] *[";" parameter]
## Suffix
+xml, +json, +ber, +der, +fastinfoset, +wbxml, +zip, +gzip
